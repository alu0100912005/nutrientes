Descripcion

Esta practica de laboratorio se ha de realizar utilizando el lenguaje de programacion Ruby y el paradigma de Programacion Funcional. Para el desarrollo de las clases se ha de seguir la metodologia de Desarrollo dirigido por pruebas (Test Driven Development - TDD) y la herramienta RSpec. Considere la jerarquia de clases Ruby para representar alimentos y listas de practicas anteriores.
