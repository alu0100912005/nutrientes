Descripcion

Esta practica de laboratorio se ha de realizar utilizando el lenguaje de programacion Ruby y el paradigma de Programacion Orientada a Objetos. Se ha de partir de la estructura de la gema generada con la herramienta Bundler y realizar el control de versiones con git siguiendo su filosofia de ramas. Para el desarrollo de las clases se ha de seguir la metodologia de Desarrollo dirigido por pruebas (Test Driven Development - TDD) y la herramienta RSpec.
